Custom Scrollbar Styling

A production-ready utility to customize scrollbars natively across WebKit/Blink (Chrome, Safari, Edge) and Gecko (Firefox) engines using simple CSS custom properties.

API Tokens & Classes

--ease-scrollbar-track
Color of the scrollbar track track background.
--ease-scrollbar-thumb
Color of the scrollbar sliding thumb.
--ease-scrollbar-thumb-hover
Color of the scrollbar thumb in hovered state (WebKit).
--ease-scrollbar-radius
Corner curves radius for WebKit track/thumb segments.
